# Анализ раскладки `ergonaut_one(backup).keymap`

Этот файл содержит **базовую (стандартную)** конфигурацию для клавиатуры Ergonaut One. Это "чистая" версия, с которой обычно начинают настройку.

⚠️ **Важное отличие:** В отличие от вашего активного файла `ergonaut_one.keymap`, этот файл **НЕ содержит** макросов, комбо, сложных Home Row Mods или специфических настроек для macOS.

## 1. Структура конфигурации

Это классическая раскладка ZMK с использованием "Tri-Layer" логики.

### Условные слои (Conditional Layers)
Реализована стандартная логика:
*   Если включен слой 1 (`LWR`) **И** слой 2 (`RSE`) -> активируется слой 3 (`ADJ`).

### Тайминги
```dts
&lt { quick-tap-ms = <200>; };
&mt { quick-tap-ms = <200>; };
```
Установлен таймаут 200мс для быстрого повторного нажатия (чтобы клавиши-модификаторы срабатывали как обычные клавиши при быстром наборе).

## 2. Разбор Слоев

В файле определено 4 слоя:

### Слой 0: `default_layer` (MAIN)
Стандартная QWERTY раскладка.
*   **Модификаторы:** Расположены по краям (`LCTRL`, `LALT`, `LGUI` на левой части; `RCTRL`, `RALT`, `LGUI` на правой).
*   **Большие пальцы (Thumb Cluster):**
    *   `&lt RSE TAB`: Tab / Слой 2 (Raise)
    *   `&mt LSHFT SPACE`: Пробел / Shift
    *   `&lt LWR RET`: Enter / Слой 1 (Lower)
    *   `&lt LWR ESC`: Esc / Слой 1 (Lower)
    *   `&mt RSHFT BSPC`: Backspace / Shift
    *   `&lt RSE DEL`: Delete / Слой 2 (Raise)

### Слой 1: `lower_layer` (SYM)
Активируется удержанием клавиши `Lower`.
*   **Верхний ряд:** Функциональные клавиши F1-F12.
*   **Средний ряд:** Символы (`!`, `@`, `#` и т.д.) и скобки.
*   **Нижний ряд:** Цифры 1-0.

### Слой 2: `raise_layer` (NUM)
Активируется удержанием клавиши `Raise`.
*   **Правая рука:** Numpad (цифровая клавиатура) + математические знаки.
*   **Навигация:** Стрелки (`LEFT`, `DOWN`, `UP`, `RIGHT`), `HOME`, `END`, `PG_UP`, `PG_DN`.
*   **Медиа:** Громкость (`VOL_UP`, `VOL_DN`, `MUTE`).

### Слой 3: `adjust_layer` (ADJ)
Активируется одновременным нажатием `Lower` + `Raise`.
Служебный слой для управления клавиатурой:
*   **Bluetooth:** Переключение профилей (`BT_SEL 0-4`), сброс подключения (`BT_CLR`).
*   **Система:** Перезагрузка (`sys_reset`), режим загрузчика (`bootloader`).
*   **ZMK Studio:** Разблокировка студии (`&studio_unlock`).

## 3. Заключение

Этот файл представляет собой **эталонную** конфигурацию Ergonaut One.
*   Если ваши эксперименты с основным файлом приведут к ошибкам или неудобствам, вы всегда можете скопировать содержимое этого файла в `ergonaut_one.keymap`, чтобы вернуть клавиатуру к заводским настройкам.
